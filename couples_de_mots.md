# Couples de mots « qui se ressemblent en dessin »

> **Idée** : chaque ligne est un **couple de mots** dont le *dessin simplifié* (style icône / pictogramme / bonhomme-bâton) se ressemble énormément. Parfait pour un jeu de type **dessin & devinette**, **piège visuel**, **« devine le bon mot »** ou variante *Undercover*.
>
> Lecture : `Mot A / Mot B` — les deux donnent (à peu près) le même croquis minimaliste.
>
> **283 couples**, regroupés par **forme** ou **famille** pour qu'on retrouve facilement un piège.

## Comment l'utiliser

- **Jeu de dessin-piège** : un joueur tire un couple, dessine *un* des deux mots de façon simple ; les autres doivent deviner lequel des deux c'est (le piège vient de la ressemblance).
- **Variante « mot caché »** : la majorité reçoit le mot A, l'imposteur le mot B (très proche). Comme les dessins se ressemblent, l'imposteur peut se fondre dans la masse.
- **Échauffement Pictionary** : idéal pour des manches rapides où le dessin minimaliste suffit.

> Astuce : les couples des sections **2 (croissant)**, **3 (objets longs)**, **12 (ciel)** et **19 (formes)** sont les plus « traîtres » — leur croquis est quasi identique.
